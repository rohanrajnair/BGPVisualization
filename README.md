# BGPVisualization

### Starting up the Webpage Locally:

Make sure [Node.js](https://nodejs.org/en/) is installed!

```
 cd .\cytoscape_graph\bgpvizapp\
 npm install
 node .\index.js
```
App url: http://localhost:3000/
